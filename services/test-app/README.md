# FastAPI application test-app

## 🚀 How to run locally

### With command line

Install dependencies:

```bash
pnpx nx run test-app:install
```

Activate virtual environment created at `test-app/.venv` and run the FastAPI server

```bash
uvicorn test-app.main:app --reload
```
